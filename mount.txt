mount
    (*https://jhnyang.tistory.com/12 으로부터 발췌)
    -디스크를 특정 디렉터리에 마운트하는 명령어

    -사용 형식
        mount [option] [device] [directory]
        option : 옵션
        device : 장치명
        directory : 마운트 할 디렉터리
    
    * 옵션
        -a  : /etc/fstab에 명시된 파일 시스템을 마운트할때 쓰임
        -t [fs_type] : 파일 시스템의 유형을 지정하는 옵션. 아무것도 적지 않으면 /etc/fstab 파일을 참조함.
        -o [추가 설정 항목] : 마운트할때 추가적인 설정을 적용할 때 사용하는 옵션. 다수의 조건을 적요앟ㄹ 대는 콤마(,)로 구분하면

        -o 옵션의 주요 항목 :
            ro  : read only, 읽기 전용으로 마운트
            rw  : read and write, 읽고 쓸 수 있게 마운트 
            remount : 해당 파티션을 다시 마운트함. 주로 '파티션의 정보를 바꿨을때'사용됨.
            loop    : loop 디바이스로 마운트 할때 사용함. (CD-ROM 이미지 파일이 ISO를 마운트해서 사용할 때 쓰임)
            noatime : 파일의 내용을 읽게 되면 access time이 변경되는데, 이 옵션을 사용하면 파일이 변경되기 전까지는 access time 이 변경되지 않기 때문에, 이 옵션을 사용해서 여러 장치의 여러 파일을 읽기만 하고 다니는 상황에서, 시스템의 처리 작업 양을 줄여 하드웨어의 성능 향상을 기대할 수 있습니다.
    
    *예시
        
        mount /dev/sdb1 /drv
        (* )

        mount -t ext4 -o ro /dev/sdb1 /mnt
        (* ext4 라는 파일 시스템을 사용하고, 읽기 전용(ro)으로 /dev/sdb1 이라는 장치를 열어서 /mnt에 마운트한다.)
