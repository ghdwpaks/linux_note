yum
    -Internet을 이용한 자동 패키지 관리
        >반드시 Internet이 연결되어 있어야 사용 가능

    -의존성 오류를 알아서 처리

    -사용 형식
        you [옵션] {install | update | remove} 패키지명
            install : 설치
            update  : 업데이트
            remove  : 삭제
    
    *옵션
        y : y/N 물어볼 때 자동으로 y 입력하여 진행

    예시 :
        yum -y remove vim-*
        'vim-'이 패키지 이름 앞에 붙은 경우 전부 삭제하기

        yum -y install vim-*
        'vim-'이 패키지 이름 앞에 붙은 경우 전부 다운받기

        yum -y update
        모든 패키지 업데이트
